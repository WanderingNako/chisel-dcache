# Dcache
## Dependencies
### JDK 11 or newer
Java 11 or later LTS releases is recommended. You can install the JDK as your operating system recommends, or use the prebuilt binaries from [Adoptium](https://adoptium.net/).
## Getting Started
To run all tests in this design
```
make test
```
To generate Verilog
```
make verilog
```
